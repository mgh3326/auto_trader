## 이미지 제작 팁

### 권장 이미지 타입

1. **플로우차트/다이어그램**
   - draw.io, Figma, Excalidraw 사용
   - 깔끔한 선과 박스
   - 한글 폰트: 나눔고딕, 프리텐다드

2. **코드 실행 스크린샷**
   - VS Code 다크 테마
   - 터미널 출력 결과
   - 중요 부분 하이라이트

3. **차트 이미지**
   - TradingView 스크린샷
   - 또는 matplotlib로 직접 생성
   - 이동평균선, RSI 표시

4. **비교 인포그래픽**
   - Canva 또는 Figma
   - 두 API 비교를 시각화
   - 아이콘 활용

### 이미지 스타일 가이드

- **크기**: 1200x600 ~ 1200x800px
- **포맷**: PNG (투명 배경) 또는 JPG
- **색상**:
  - 주요색: #4A90E2 (파란색)
  - 강조색: #F5A623 (주황색)
  - 배경: #FFFFFF 또는 #F8F9FA
- **폰트**:
  - 제목: 18-24px Bold
  - 본문: 14-16px Regular

### 필수 포함 요소

1. **시스템 아키텍처 이미지**
   ```
   [KIS API] → [데이터 수집] → [기술적 지표 계산] → [프롬프트 생성] → [Gemini AI] → [투자 판단]
   ```

2. **프롬프트 구조 다이어그램**
   ```
   ┌─────────────────┐
   │  헤더 (종목명)   │
   ├─────────────────┤
   │ 기술적 지표 요약 │
   ├─────────────────┤
   │  기본 정보      │
   ├─────────────────┤
   │  가격 지표      │
   ```
